# Cucco プレイ用クライアント

実際のゲームプレイに近い体験を目指したブラウザクライアント。卓を囲む
レイアウト(自席が常に下中央)で、カードの配布・交換・チップの移動・
宣言などのアクションがアニメーションで可視化される。

- 起動方法・共有URL・運用手順: `docs/web-client-operations.md`
- プロトコル/共有モジュール: `clients/web-common/`(リファレンス
  `clients/web/`と共通。プロトコル変更時は両クライアントを更新する)
- 実装の骨子: `src/gameState.js`(公開状態ミラー)→ 意味オペレーション →
  `src/anim/queue.js`(プレゼンテーションキュー)→ `src/scene/table.js`
  (保持DOMのテーブルシーン)。**自分宛のプロンプト受信時はキューを
  即時完了させる**(サーバーのタイムアウトは演出を待たないため)

カードのイラスト・アイコンは使用しない(名称タイポグラフィのみ)。
