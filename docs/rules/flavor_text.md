# フレーバー文言

特殊札が発動した際の宣言文言。日本語版を正式採用する。UI・プロトコルのメッセージ文言の参考にする。

## 正式採用: 日本語宣言セリフ

| 札 | 宣言文言 |
|---|---|
| クク(梟) | 「私は全てを明らかにする」 |
| 人間 | 「私に挑むものは全て滅びるであろう」 |
| 馬 | 「私は跳ねて飛び越える」 |
| 猫 | 「私の呪いはさかのぼる」(拒否時は「ニャー」) |
| 家 | 「あなたは私を通り過ぎる馬と同様です」 |
| 道化 | 「私は最も弱く、最も強い」 |

出典: dice.saloon.jp「カンビオ」ルール記事(`docs/rules/cambio_dice-saloon.txt`)

## 参考情報: イタリア語原題(名古屋大学OCWスライドより、文字情報のみ)
