# プレイサマリー(グランペール版 付属)

グランペール版Cuccoに付属していたプレイサマリーを画像から書き起こし。カードの戦術的判断の目安として、AIプレイヤーの基礎戦略設計に参考にする。イラストは参照していない(文字情報のみ)。

## カードごとのチェンジ判断表

| カード名 | チェンジの判断 | 戦術ポイント | チェンジを受けた時の対応 |
|---|---|---|---|
| クク | ノーチェンジ(1番手は温存、2番手以降は即宣言) | ククは最強。温存して状況に応じて宣言、宣言すると即ゲーム決着 | チェンジは強制、チェンジ後も「クク」の宣言権は維持 |
| 人間 | ノーチェンジ | チェンジ要求が来たらカードを見せて失格にできるため強力、ノーチェンジ確定 | チェンジ要求者を失格にできる |
| 馬 | ノーチェンジ | 交換拒否できるので、強い防御、ノーチェンジ確定 | チェンジ要求は拒否(パス)可能 |
| 家 | ノーチェンジ | 交換拒否できる場面、ノーチェンジ確定 | チェンジ要求は拒否(パス)可能 |
| 猫 | ノーチェンジ | 交換時に、相手か元持ち主が失格になる攻撃的なカード。ノーチェンジ | チェンジ時、相手か元カード所有者のいずれかが失格になる |
| 数字 | 下の表を参照 | 下の表を参照 | 下の表を参照 |
| 桶 | チェンジ | 弱いカード、チェンジ必須 | 特に対応なし |
| 仮面 | チェンジ | 弱いカード、チェンジ必須 | 特に対応なし |
| ライオン | チェンジ | 弱いカード、チェンジ必須 | 特に対応なし |
| 道化 | チェンジ(即回避) | 最弱カード、入手時に失格なので絶対チェンジ | 交換で手に入れると即失格 |

## 数字カードのチェンジ基準(人数目安、要確認)

| 人数 | チェンジ基準(目安) | 説明 |
|---|---|---|
| 3人以下 | 7以下チェンジ | 弱いカードが引っかかりやすく、負けるリスクが高いため、ある程度の強さまで上げたい |
| 4〜6人 | 6以下チェンジ | 負ける可能性が少し減るので、少し弱くてもノーチェンジ |
| 5人 | 5以下チェンジ | 中間的なカードでも他にもっと弱い人がいる可能性が高まる |
| 6人 | 4以下チェンジ | 少し弱いカードでも様子見可能。むやみにチェンジすると後回しにリスク増 |
| 7人 | 3以下チェンジ | 多少弱くても、他に最弱がいる可能性が高い、安易なチェンジが不利になることも |

**注**: 人数区分(3人以下→4〜6人→5人→6人→7人)に重複・順序の乱れがあるように見える。画像からの書き起こし時点では未確認、将来的に要検証。
