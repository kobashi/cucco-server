# ルール調査メモ(作業中)

最終ルールはユーザーから別途提示される。このメモは資料収集段階での確定事項・未解決事項の記録。

## 確定した仕様

- **カード構成**: 22種類×2枚 = 44枚
  (道化, 獅子, 仮面, 桶, 0, 1〜12, 家, 猫, 馬, 人間, クク の22ランク)
  ゼミのランク早見表(No1〜22)と一致。40枚版(PDF/dice.saloon.jp記載)は不採用。
- **チップ支払い**: 敗者はディール数に応じて増える枚数を支払う(1ディール目=1枚, 2ディール目=2枚, 3ディール目=3枚, ...)。
  4ディール目以降は失格者がゲームから脱落(チップ支払いなし)。
  sealion版の「毎ディール一律1枚」は不採用。
- **手番の流れ**: 親が全員に1枚配布→親の右隣から反時計回りに、右隣のプレイヤーとの交換要否を選択→最後(親)は山札の上と交換要否を選択→全員オープンし最弱のカードの人が負け。
- **特殊札の効果**(全資料で一致):
  - 道化: 交換で受け取った人が即失格。山札から引かれた場合はクク(最強)を上回る最強札になる。
  - 家・馬: 交換要求を拒否でき、要求はさらに右隣のプレイヤーに移る。山札から引かれた場合は次の山札カードに進む。
  - 猫: 「ニャー」で拒否し、要求者が現在持っているカードの最初の持ち主(元の所有者)が失格。山札から引かれた場合も同様の判定。
  - 人間: 交換要求を拒否し、要求した人を即失格にする。山札から引かれた場合は親が失格。
  - クク: いつでも「クク」と宣言してディールを即終了できる。山札から引かれた場合は交換前のカードのまま(拒否扱い)。
  - 数字札(0〜12)・獅子・仮面・桶: 特殊効果なし、交換は拒否できない。
