# ククの作戦について考慮すること

出典: ゼミ内Googleスプレッドシート「ヒント」タブ (2026-07-08時点)

- ククの作戦ついて考慮すること
- チェンジ・ノーチェンジを判断するだけ
- チェンジのリスク判定 / 交換すると失格させられるカードの存在
- チェンジの必要性判定 / このラウンドで自分のカードが最弱となりえるかの判断
- 残り人数
- 場に積まれたチップの枚数
- ククの宣言のタイミング
- 自分のカードの強さ
- 自分の順番が回ってくるまでの、プレイヤーの行動
- ノーチェンジの回数
- チェンジの回数（チェンジ、ノーチェンジのパターン発生）
- 既に使用されたカードの内容
- 親までチェンジが続き、親が山札をめくった後に最弱のカードとなり得るかどうか
- 親の場合、引き直したカードで状況が改善されるかどうか
- ラウンド数（復活有のラウンドか、無しのラウンドか）
